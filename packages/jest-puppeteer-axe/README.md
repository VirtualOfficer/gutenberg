# Jest Puppeteer Axe

[Axe](https://www.deque.com/axe/) (the Accessibility Engine) API integration with [Jest](https://jestjs.io/) and [Puppeteer](https://pptr.dev/).

## Installation

Install the module

```bash
npm install @wordpress/jest-puppeteer-axe --save-dev
```
